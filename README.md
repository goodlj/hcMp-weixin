### 微信小程序版本说明

微信小程序1.0.0：
 ### 功能描述：
				1.新增门店;
				2.用户可查询全部订单;
				3.全部订单可进行时间(近7天，近1个月，近3个月，近6个月，今年)筛选;
				4.行业动态资讯展示;
				5.监管码查询;
				6.电子随货、电子质检下载;
